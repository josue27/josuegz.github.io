    Project that consisted in realizing a "ride" in v.r. for an amousement park called "Kataplum". There where tons of challenges, one of them was integrating a hand tracking system with the Leap Motion along side withe HTC Vive Pro so the user could enjoy the ride without using controlls. Also there were 32 VR stations that must be controlled via LAN in order to sync up the ride. I programmed everything and also implemented the interaction.
